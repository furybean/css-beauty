# CSS Beauty

> A umd package for making your css beauty.

## Usage

Node.js:

```Bash
npm i css-beauty --save
```

Browser:

```HTML
<script src="https://unpkg.com/css-beauty@0.2.0"></script>
```

```JavaScript
const formatted = CSSBeauty.beauty('body { font-size: 12px; }');
document.write('<pre>' + formatted + '</pre>');
```
